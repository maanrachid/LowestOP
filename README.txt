# Find Lowest Operator
-----------------
This is a test for senior software engineers.



## Prerequisites
------------------
- Ensure you have the `g++` compiler installed to run this code.



## Steps
-------------
1. Clone the repository:
    
    git clone https://github.com/maanrachid/LowestOP
    
2. Navigate to the `LowestOP` directory:
   
    cd LowestOP
    
3. Compile the code:
    g++ -std=c++11 -o low_price low_price.cpp price_checker.cpp
    

4. To run the code:
    
    ./low_price
    


## Unit Tests
----------------
To run the unit tests:
1. Compile the test file:
    
    g++ -std=c++11 -o Test Test.cpp price_checker.cpp
    

2. Execute the unit tests:
    
    ./Test
